<summary><b>🔐 Qsafe — Quantum-Safe Communication Protocol</b></summary>
<br/>

An open-source scaffold for a **post-quantum secure communication stack**. It pairs quantum-resistant key exchange and signatures with a safe fallback to classical TLS, verifies both sides of a connection with hardware-backed attestation before any session starts, and carries every message inside an encrypted, transport-agnostic envelope with automatic key rotation — backed by built-in threat modeling, fuzzing, and CI security scanning.

## 🌍 Why Open Source Matters to Me

Open source gives students like me the opportunity to learn beyond classrooms, collaborate with real developers, and build projects that can create impact.

As an undergraduate from Sri Lanka 🇱🇰, I want to help more students become confident with:

- Git and GitHub
- Clean commit history
- Pull requests and collaboration
- Documentation
- Issue tracking
- Responsible open-source contribution
- Building projects that are useful, readable, and maintainable

I believe a good developer is not only someone who writes code, but someone who can **communicate, document, collaborate, and maintain**.
